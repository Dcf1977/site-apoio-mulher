# Rompendo o Silêncio 💜

**Portal de Apoio e Fortalecimento à Mulher**

O projeto **Rompendo o Silêncio** é uma plataforma web desenvolvida para oferecer acolhimento, informação e recursos para mulheres em situação de vulnerabilidade ou violência doméstica. O objetivo principal é transformar a informação em uma ferramenta de salvamento, conectando vítimas aos canais oficiais de denúncia.

## 🚀 Funcionalidades Principais

- **Botão de Saída Rápida (Pânico):** Funcionalidade de segurança que permite fechar ou redirecionar a página rapidamente para proteger a privacidade da usuária.
- **Questionário de Identificação:** Uma ferramenta interativa para ajudar a identificar sinais sutis de violência psicológica, patrimonial e física.
- **Acessibilidade (VLibras):** Integração com o plugin VLibras para garantir que mulheres com deficiência auditiva tenham acesso total ao conteúdo.
- **Guia de Serviços:** Informações diretas sobre a Lei Maria da Penha e contatos de emergência (Ligue 180).
- **Formulário de Relato:** Canal seguro e sigiloso para envio de mensagens e pedidos de ajuda.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica das páginas.
- **CSS3:** Estilização moderna com foco em legibilidade e fontes do Google (Inter).
- **JavaScript:** Lógica do questionário, barra de progresso e funções de segurança.
- **Formspree:** Integração para recebimento dos relatos via formulário.
